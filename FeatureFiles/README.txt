This Folder will contain the Files Feature_Descriptors.mat and FeaturesBagofVisualVocabulary.mat that are computed after executing the codes BuildVisualVocabulary.m  and Train_BagFeaturesVisualVocabulary.m


Feature_Descriptors.mat file will conatin information about the 3D SIFT descriptors and class labels of the video files and is neccessary for the Bag of Visual Words process.


FeaturesBagofVisualVocabulary.mat will contain the feature vectors obtained by the Bag of Visual Words model and the class labels of the video files.
